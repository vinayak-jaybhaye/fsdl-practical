# FSDL Final Mock Practical Solutions

This repository contains one folder per problem statement (`01` to `30`).

## Quick Start (Frontend Tasks)
Open any `index.html` file from tasks `01` to `28` in a browser.

## Quick Start (Backend + MongoDB Tasks)

### Task 29 - Student Record Management
- Path: `29-student-record-management-mongodb`
- Commands:
  - `npm install`
  - `npm start`
- URL: `http://localhost:4001`

### Task 30 - Event Registration System
- Path: `30-event-registration-system`
- Commands:
  - `npm install`
  - `npm start`
- URL: `http://localhost:4002`
